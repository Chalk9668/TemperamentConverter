# 改弦更張（仮） - TemperamentConverter
改弦更張といいます。名前は中国の故事に基づいています。UTAUでノートをtフラグを使って所定の調律に書き換えるプラグインです（仮）。ワイがモアサプ民なのでMoreSamplerを想定してください。システムにPythonを利用しているから、もしかしたらインタプリタが必要かも。精度は謎。

現状1/4commaMeantoneTemperamentを31EDOで鳴らした時、Cから5L3SでMOSを生成した時の音程のみ対応しています（伝われ）。自由に変位は可能。一応12EDOに戻せたり。ウルフの調節には未対応。

更新する時？ワイが必要になったときだ。Vocalsifterで調教するよりはUTAU上でやったほうがラクだろうし、時間対効果とワイの微分音音楽技能と相談。

やりたいこと（優先度順）
- マジックナンバーじゃなくてちゃんと音律計算して出力したい（ありがとうAnacondaやっぱPython愛してる）
- VocalSifterみたいなユーザ定義音律みたいな機能もつくりたい
- 31EDO以外のnEDO、様々な不等分律なども使えたい
- .scl/.kbm や .tun 等の調律ファイルや調律情報をつかえるようにしたい
- おリッチなUIに
